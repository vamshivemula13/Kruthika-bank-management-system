# 🏦 Kruthika Bank Management System

## 📌 Project Overview

Kruthika Bank Management System is a simple web-based banking application developed to simulate basic banking operations. The system allows users to create bank accounts, deposit and withdraw money, check account balances, and manage account records through an interactive dashboard.

This project was developed to practice frontend development concepts such as DOM manipulation, event handling, and CRUD operations using JavaScript.

---

## 🚀 Features

* Create a new bank account
* Deposit money into an account
* Withdraw money from an account
* Check account balance
* View all accounts in a table format
* Delete an account
* Data persistence using Local Storage
* Clean and responsive user interface

---

## 🛠 Technologies Used

* **HTML** – Structure of the application
* **CSS** – Styling and layout
* **Bootstrap** – Responsive design
* **JavaScript** – Application logic and functionality
* **Local Storage** – Data persistence in browser
